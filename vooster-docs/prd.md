## BD 한의원 웹사이트 프로젝트 개요

BD 한의원 웹사이트를 Apple 스타일의 현대적 디자인으로 완전히 리뉴얼하였습니다. Next.js 15, React 19, TypeScript 기반으로 구축하여 SEO 최적화, 고급 인터랙션, 반응형 UI를 구현했습니다. 치과에서 한의원으로 브랜딩을 완전히 전환하여 침구치료, 한약처방, 추나요법, 체질진단 등 한의학 전문 서비스에 특화된 웹사이트로 제작되었습니다.

## 주요 달성 성과

### 디자인 시스템
*   **Apple 스타일 디자인 시스템 완성**: SF Pro 폰트, 시스템 컬러(시스템 블루, 그레이 팔레트), 정밀한 타이포그래피 시스템 적용
*   **대형 히어로 섹션**: 비디오 배경과 대형 타이포그래피로 임팩트 있는 첫 인상 제공
*   **인터랙티브 요소들**: 호버 애니메이션, 모션 그래픽스, 마이크로 인터랙션

### 고급 인터랙션
*   **Framer Motion 애니메이션**: 페이드인, 슬라이드인, 스크롤 애니메이션
*   **패럴랙스 스크롤링**: 배경 이미지와 콘텐츠의 다른 속도로 움직이는 시각적 효과
*   **이미지 시퀀스 애니메이션**: 스크롤에 따라 자연스럽게 변하는 이미지
*   **비디오 배경 및 모달**: 고품질 비디오 플레이백 및 모달 지원

### 한의학 전문 콘텐츠
*   **침구치료**: 무통 침술, 전자침 시스템 소개
*   **한약처방**: 체질 맞춤형 한약, 개인 맞춤 처방
*   **추나요법**: 수기 요법, 척추 교정 칙료
*   **체질진단**: 과학적 체질 분석 및 맞춤 칙료 계획

### 기술적 성취
*   **완전 반응형 디자인**: 모바일/태블릿/데스크탑 모든 해상도에서 최적화
*   **SEO 최적화**: Next.js 15 기반 SSR/SSG, 메타데이터 완비
*   **성능 최적화**: 이미지 최적화, 코드 스플리팅, 레이지 로딩

## 구현된 사이트 구조

### 현재 구현 완료 페이지
*   **메인 페이지** (`/`): BD 한의원 소개 및 브랜드 홍보
  - Apple 스타일 히어로 섹션
  - 한의학 서비스 소개 (침구치료, 한약처방, 추나요법)
  - 현대 한의학 기술 섹션
  - FAQ 섹션

*   **진료과목 페이지** (`/services`): 전문 한의학 칙료 소개
  - "왜 BD 한의원인가?" 히어로 섹션
  - 한의원 특징 및 강점 소개
  - 7단계 체계적 진료 과정 (초진상담 → 침구칙료 → 한약처방 → 추나요법 → 사후관리)

### 향후 확장 계획
*   체질진단 전문 페이지
*   커뮤니티 (공지사항, 칙료 사례)
*   예약 시스템 연동
*   온라인 상담 시스템

## 기술 스펙

*   프론트엔드:
    *   Next.js 13 (App Router): SSR/SSG 기반 SEO 최적화
    *   React 18 + TypeScript: 모듈형 컴포넌트 개발
    *   styled-components: CSS-in-JS 기반 스타일 관리 및 다크모드/반응형 테마 지원
    *   React Query: 데이터 fetching 및 캐싱, 추후 API 연동 확장 가능
*   콘텐츠 관리:
    *   JSON 기반 데이터 관리
    *   각 페이지/컴포넌트별 콘텐츠(JSON 파일) 분리
    *   텍스트/이미지/링크는 JSON을 수정하는 것만으로 교체 가능
    *   React 컴포넌트 Props 주입 방식
    *   JSON 데이터를 Props로 전달해 UI 렌더링
    *   HTML 렌더링
    *   단순 텍스트뿐 아니라 `<strong>`, `<ul>`, `<br/>` 등 HTML 태그 포함 가능
    *   React 컴포넌트에서 dangerouslySetInnerHTML 또는 ReactNode 타입 Props 활용
    *   하드코딩 ❌, JSON/HTML 교체 ✅
*   테스트:
    *   Jest + React Testing Library
    *   단위 테스트 (네비게이션, FAQ 아코디언, 슬라이더)
    *   통합 테스트 (페이지 전환, 반응형 렌더링)
    *   접근성 테스트 (alt 속성, role, 키보드 네비게이션)

## SEO & 성능 최적화

*   SEO
    *   페이지별 `<title>`, `<meta description>`

    *   JSON-LD (LocalBusiness, FAQPage schema)

*   성능 최적화

    *   코드 스플리팅 + lazy-loading
    *   외부 위젯(Google Map 등) 지연 로딩
    *   Lighthouse 기반 성능/SEO 점검
